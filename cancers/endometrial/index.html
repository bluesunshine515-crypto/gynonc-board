📌 v2 更新（NotebookLM 對 NCCN v.2.2026 review）

  • 🚨 IA1 / IA2 對調修正：IA1 = confined to endometrium（無肌層侵犯）；IA2 = <50% 肌層侵犯（v1 寫反）
  • IC aggressive histology 治療放寬：non-invasive serous/clear cell 可選 observation 或 vault brachy，不一律 chemo
  • NRG-GY018 / RUBY Cat 1 擴大：適應症含 stage III-IV primary / adjuvant（不只 advanced/recurrent）
  • DUO-E 分級：dMMR Cat 1，pMMR Carbo/Pac/Durva + Durva/Olap maintenance Cat 2B
  • Lenva+Pembro Cat 1：pMMR、platinum-failed（含 neoadjuvant + adjuvant 都算）
  • HER2 testing 擴大：所有 p53abn + 所有轉移/復發；T-DXd Carcinosarcoma IHC 1+/2+/3+ 皆可（一般只 2+/3+）
  • Carcinosarcoma 例外：CS first-line 用 Carbo/Pac alone ± Trastuzumab，Pembro 不適用
  • ER+ hormone：低惡性度/緩慢/低瘤量 ER+ recurrent 可選 Megestrol / Letrozole+Palbociclib 為一線
