# gynonc-board

> 婦癌專科醫師考試準備 — NCCN / FIGO / TSGH 指引摘要 + 題庫 + Landmark trials

線上版：https://bluesunshine515-crypto.github.io/gynonc-board/

## 範圍

六大婦癌：子宮頸、子宮內膜、卵巢、外陰、陰道、滋養細胞 (GTD)

每癌別含：
- **分期速查**（FIGO 2018/2023）
- **處置流程**（NCCN 摘要 + 決策樹）
- **藥物 cheatsheet**（TSGH 治療處方）
- **Landmark trials**（含 INTERLACE、KEYNOTE-826 等近年新試驗）
- **練習題**（沿用 Obstetrica 題庫格式 + 自製專科考題）

## 技術

- 純靜態 HTML + JS（無 backend）
- localStorage 存個人進度
- GitHub Pages 部署
- 題目資料 JSON 格式（人類可讀、可版控）
